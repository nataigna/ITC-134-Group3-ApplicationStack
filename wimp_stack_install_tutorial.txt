WIMP stack installation Tutorial


Aggie Wheeler Bateman, Liyun Cecil, and Nataliia Carlton

This guide will help a user install the WIMP application stack on Google Cloud Platform to run a dynamic web site. The WIMP application stack is composed of Windows as the operating system, Internet Information Services as the web server, MariaDB as the database, and Python as the programming language. Some of the software is open source, and hence, it is partially free.(https://en.wikipedia.org/wiki/WIMP_(software_bundle))
