Django Stack Installation Guide

Coauthored by Aggie Wheeler Bateman, Liyun Cecil, Nataliia Carlton

This is a guide that helps you install a Django web application stack on a Google Cloud Services virtual machine. This particular guide includes Ubuntu as the operating system, Apache as the web server, Python/Django as the programming language, and PostgreSQL as the database.

The following links will take you to Google Documents with step by step instructions:

  • Ubuntu (Operating System) and Apache (Web Server) Installation
  • Django/Pip/Python Installation (Programming Language)
  • PostgreSQL Install (Database)
  • Git Install (Version Control)

All of these guides include referenced sources in case you need additional information or a different explanation of a section.
